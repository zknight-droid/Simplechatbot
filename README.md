## About The Project

Chat Bot sederhana yang memungkinkan pengguna dapat berinteraksi dengan bot melalui terminal, bot ini menggunakan file dataset.txt sebagai responsenya



### Built With

* PYTHON




<!-- GETTING STARTED -->
## Getting Started

Project ini menggunakan Python versi 3.10.9.

### Prerequisites

Pastikan versi pip yang terinstal adalah yang terbaru.
* pip
  ```sh
  pip install --upgrade pip
  ```

### Installation

1. Clone repositorinya
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Pindah ke direktori Simplechatbot
   ```sh
   cd Simplechatbot
   ```
3. Buatlah virtual environment
   ```sh
   python3 -m venv myenv
   ```
4. Jalankan virtual environment
   ```js
   myenv\Scripts\activate.bat
   ```
4. Install library yang diperlukan
   ```js
   pip install -r requirements.txt
   ```
